# Lead Criteria & Scoring Rubric

*This file defines what a good Marcello lead looks like. Any agent doing prospect research should match new leads against this pattern and score them with this rubric. The worked examples live in `lead-scoring-examples.csv` (supplied by Christian, 2026-07; re-verify all contact details and business activity before outreach).*

## The target pattern

**Brands:** founder-led fashion, jewelry, accessories, or lifestyle brand with strong product and imagery, roughly **$500K–$15M annual sales**, whose website no longer matches its ambition. The tell is a *gap*: editorial/runway/press momentum (NYFW slot, CFDA/LVMH recognition, Vogue coverage) on one side, a structurally minimal or template-grade site on the other. Founder must be publicly reachable (named, with a business email).

**PR partners:** boutique fashion/luxury communications agencies whose own promise is senior, high-touch, no-junior-handoff service — the same model Marcello sells for digital. These are **referral/white-label partnerships, not one-off site sales**: one relationship can produce repeated launches and redesigns. Indicative project value is $0 — the value is the pipeline.

## Record format (columns for every lead)

Lead Type · Company · Decision Maker · Public Business Email · Website · Geography · Why It Fits · Visible Trigger / Momentum · Digital Opportunity · Suggested First Offer · Indicative Project Value · Priority (1–5) · Fit · Trigger · Need · Access · Budget · Leverage · Weighted Score · Outreach Hook · Next Research Step · Last Verified · Primary Source · Supporting Source

## Scoring dimensions (each 1–5)

| Dimension | Question it answers |
|---|---|
| **Fit** | Does the brand match the target pattern (founder-led, fashion-adjacent, right size)? |
| **Trigger** | Is there a visible, timely momentum event (show, prize, press) to hang outreach on? |
| **Need** | How large is the gap between brand quality and current digital presence? |
| **Access** | Is the decision maker directly reachable (named founder + public email)? |
| **Budget** | Can they plausibly pay (price points, distribution, funding/prize signals)? |
| **Leverage** | Does winning this lead create more leads (referrals, portfolio value, partnership pipeline)? |

**Weighted Score** blends the six (observed range in examples: 3.7–4.7; PR partnerships score highest on Leverage). Work leads in descending weighted-score order within each type.

## Mapping scores to the offer ladder (see business-plan.md)

- Elegant-but-static site, or unclear what to rebuild → **Digital Direction Sprint** ($2.5k–$3.5k)
- New/small catalog, strong story, thin store → **Launch Site** ($8k–$12k)
- Real store + real momentum + clear gap → **Signature Commerce** ($18k–$28k)
- Established brand / ambitious relaunch → **Digital Flagship** ($35k–$75k)
- PR agency → **Referral / white-label partnership** (relationship-first, no cold project pitch)

## Rules

1. Never criticize an elegant site — lead with a high-value *feature* concept instead (see Colleen Allen example).
2. Every lead needs a Next Research Step completed before outreach.
3. Last Verified must be a real date; anything older than a season gets re-verified.
4. Cross-check against `prospects.csv` before adding — no duplicates, update in place.
