# Outreach Templates

Rules before sending anything:

1. **Eyeball the prospect's site in a browser first** — confirm the weakness is real and current.
2. **Get HAITCH's written OK** before naming them.
3. Every email is rewritten per prospect from the hook in `prospects.md`. If you can't personalize it in one honest sentence, don't send it.
4. ≤120 words, plain text, no attachments, no links except the portfolio. One clear ask: a short call.
5. Send from the studio address only — never a slang.ai address. 10/week max; log in `prospects.csv`.

---

## A — Brand founder (standard)

> Subject: your site vs. your clothes
>
> Hi [name] —
>
> I came across [brand] through [hook — the press piece / stockist / show]. The work is beautiful. The website isn't doing it justice — [one specific observation: "the shop page shows two pieces," "there's no way to see prices," etc.].
>
> I run Marcello, a small studio in New York building sites for fashion labels — I'm currently rebuilding the online store for HAITCH, the NYC menswear label. I work directly with founders, start to finish, and I care about the details the way you do.
>
> Would you be open to a 20-minute call? Worst case you leave with a free list of what I'd fix.
>
> Christian
> marcello.studio

## B — PR firm / showroom

> Subject: your clients' sites are better than yours
>
> Hi [name] —
>
> [Personal line from hook — the Fashionista spotlight / the Miansai win / the anniversary.] Congrats.
>
> Here's the thing I noticed: you represent brands with beautiful web presences, and [firm]'s own site is [specific: "a redirect to Instagram" / "three pages" / "a portfolio subdomain"]. Editors and prospective clients do look.
>
> I run Marcello, a studio in New York building sites for fashion — currently rebuilding HAITCH's online store. A PR firm's site is a one-to-two-week project for me, not a season.
>
> 20 minutes some time this week?
>
> Christian
> marcello.studio

## C — Urgent problem (compromised / broken / placeholder)

> Subject: [brand] site issue — flagging in case you don't know
>
> Hi [name] —
>
> Quick heads-up from a stranger: [specific issue — "your homepage is rendering injected spam links, which usually means the site's been compromised" / "your site is currently unreachable because of an SSL certificate error" / "there are placeholder products live on your store"]. No pitch attached to this part — [if compromised: "your developer should rotate credentials and audit plugins either way"].
>
> For context, I run Marcello, a web studio in New York working with fashion labels (currently rebuilding HAITCH's store). If you don't have someone to fix this, I can — quickly. Either way, wanted you to know.
>
> Christian
> marcello.studio

## Follow-up (one only, 6–8 days later)

> Subject: re: [original]
>
> Hi [name] — floating this back up. If the timing's wrong, a one-word "later" is a totally fine answer and I'll check back next season.
>
> Christian

No second follow-up. A non-answer after two touches is a "no" for this season; mark `lost` with a note and revisit in 6 months.
